#Webx是什么?
Webx是一个框架，它可用来做下面的事情：

创建一个全功能的Web应用
Webx提供了创建一个Web应用所需要的所有必要功能。

创建一个新的Web框架
Webx允许你定制、甚至重写大部分的Webx框架逻辑，从而实现全新的功能，或者和其它应用框架相整合。

创建一个非Web应用
Webx的功能并不受限于Web应用，而是对所有类型的应用都有帮助。

Webx所提供的SpringExt子框架是对Spring框架的扩展，能简化Spring的配置，加强了Spring组件的扩展性。
